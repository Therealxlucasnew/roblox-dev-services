# Roblox Dev Services Website

A static website for Roblox development commissions.

## Files

- `index.html` - Page content and layout
- `style.css` - Responsive visual styling
- `script.js` - Contact form mailto behavior

## Contact Setup

Open `script.js` and replace:

```js
const email = "your-email@example.com";
```

with the email address you want clients to contact.

## Publish With GitHub Pages

Push this folder to a GitHub repository, then enable GitHub Pages from the repository settings.
